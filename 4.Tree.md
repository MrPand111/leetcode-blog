# Tree

144 二叉树的前序遍历

- 难度：medium

- 这种题遍历的题现在感觉就是套路了，肯定得用栈或队列记录元素，唯一不同的就是推进栈or队列的顺序

``` js
var preorderTraversal = function(root) {
    if(!root) return [];
    const nodes = [];
    const result = [];
    nodes.push(root);
    while(nodes.length > 0) {
        var cur = nodes.pop();
        result.push(cur.val);
        if(cur.right) nodes.push(cur.right);
        if(cur.left) nodes.push(cur.left);
    }
    return result;
};
```

94 二叉树的中序遍历

- 难度：medium

- 和前序遍历相比一个问题就是中间节点在迭代的时候会有两次，如果不操作一下就会无限循环，所以我直接设置了当第二次被遍历到的时候直接是值而非节点，这样就能分辨出不同了

``` js
var inorderTraversal = function(root) {
    const nodes = [];
    const result = [];
    if(!root) return [];
    nodes.push(root);
    while(nodes.length !== 0) {
        const cur = nodes.pop();
        let flag = true;
        if(typeof cur === "number") {
            result.push(cur);
            flag = false;
        } else if(!cur.left) {
            result.push(cur.val);
            flag = false;
        }
        if(cur.right) {
            nodes.push(cur.right);
        }
        if(flag) {
            nodes.push(cur.val);
        }
        if(cur.left) {
            nodes.push(cur.left);
        }
    }
    return result;
};
```
