# 9.Matrix

48 旋转图像

- 难度：medium

- 第i行开始每次for循环从i开始，前面的在之前行交换过，所以不需要再交换，最好再逆序

``` js
var rotate = function(matrix) {
    const n = matrix.length;
    const hash = [];
    // 第i行从i个元素开始交换
    for(let i=0;i<n;i++) {
        for(let j=i;j<n;j++) {
            [matrix[i][j],matrix[j][i]] = [matrix[j][i],matrix[i][j]];
        }
    }
    // 倒序
    for(let i=0;i<matrix.length;i++) {
        matrix[i].reverse();
    }
    return matrix;
};
```
